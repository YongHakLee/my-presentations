## MCP(Model Context Protocol)

인공지능(AI) 모델이 외부 데이터 소스, 도구 및 시스템과 원활하게 상호 작용할 수 있도록 설계된 개방형 표준 프로토콜

## 사용 방법

- '~ MCP' 키워드 검색

![search-mcp](./imgs/search-mcp.png)

- url 접속하여 설치 방법 확인

![installation-mcp](imgs/installation-mcp.png)

- 보통 아래와 같은 형식

```JSON
{
  "mcpServers": { // MCP 서버들의 목록을 정의하는 최상위 객체
    "sequential-thinking": { // MCP 서버의 고유한 이름
      "command": "npx", // MCP 서버를 시작하기 위해 실행할 명령어, 임시 다운로드
      "args": [ //  command로 지정된 명령어에 전달될 인수(arguments)들의 목록을 배열 형태로 정의
        "-y", // npx 명령어의 옵션, 확인 절차 생략
        "@modelcontextprotocol/server-sequential-thinking" // 실제 패키지의 이름
      ]
    }
  }
}
```

- 보통 MCP Servers를 관리하는 JSON 파일에 위와 같은 형식으로 작성해놓고 사용
- 사용하는 프로그램(Cluade, Cursor, Windsurf 등)에 따라 방법이 조금씩 다를 수 있음
- CLI계열(Gemini CLI, Claude Code 등)에서 프로젝트 단위로도 추가할 수 있음
- '(MCP 이름)을 사용해서 ~을 해줘'와 같이 입력하면 알아서 사용함

## sequential-thinking

![sequential-thinking](imgs/sequential-thinking.png)

- 역할: 복잡한 문제 해결을 위한 '단계별 사고' 보조 도구
- 핵심 기능
  - 복잡한 문제를 정의, 분석, 종합, 결론 등 여러 단계로 구조화하여 사고 과정을 체계적으로 관리
  - 각 사고 단계별 진행 상황을 추적하고, 이전 생각을 수정하거나 다른 해결 경로(분기)를 탐색하는 유연한 문제 해결 지원
  - 전체 사고 과정에 대한 요약 생성 기능 제공
- 주요 용도: 명확한 절차가 필요하거나 여러 대안을 비교해야 하는 문제의 계획 및 분석

## playwright

![playwright](imgs/playwright.png)

- 역할: AI 에이전트가 웹 브라우저를 직접 제어하고 자동화하는 도구
- 핵심 기능
  - Playwright 프레임워크를 기반으로 웹페이지 탐색, 클릭, 폼 작성 등 실제 사용자 행동을 자동화
  - 스크린샷이나 시각적 분석 없이 웹페이지의 구조적 데이터(접근성 트리)를 통해 상호작용하여 높은 신뢰성 보장
  - 자동화된 테스트 케이스 생성, 웹 스크래핑, 자바스크립트 실행 등 다양한 웹 자동화 작업 수행
- 주요 용도: 반복적인 웹 기반 작업 자동화, 웹 애플리케이션 테스트, 데이터 수집

## context7

![context7](imgs/context7.png)

- 역할: AI 에이전트에게 '최신 기술 문서'를 실시간으로 제공하는 도구
- 핵심 기능
  - AI 모델의 학습 데이터에 없는 최신 라이브러리나 프레임워크의 공식 문서와 코드 예제를 제공
  - 버전별로 정확한 정보를 제공하여 AI가 오래되거나 잘못된 코드를 생성(환각)하는 문제 해결
  - 개발자가 문서 검색을 위해 IDE를 떠날 필요 없이, 프롬프트 내에서 바로 최신 정보 활용 가능
- 주요 용도: 새로운 기술이나 빠르게 변화하는 라이브러리를 사용한 코드 생성 및 디버깅

## magicUI

![magicUI](imgs/magicui.png)

- 역할: AI 에이전트가 멋진 UI 컴포넌트를 탐색하고 코드를 생성하도록 돕는 도구
- 핵심 기능
  - Magic UI 라이브러리에 포함된 애니메이션, 특수 효과 등 다양한 UI 컴포넌트에 대한 접근 제공
  - AI가 사용 가능한 컴포넌트 목록을 확인하고, 특정 컴포넌트의 소스 코드를 가져와 프로젝트에 바로 통합
  - 자연어 프롬프트만으로 복잡하고 세련된 UI를 빠르게 프로토타이핑하고 개발 가능
- 주요 용도: UI/UX 프로토타이핑, 프론트엔드 개발 생산성 향상

## chrome-devtools

![chromedevtools](imgs/chromedevtools.png)

- 역할: AI 에이전트에게 '크롬 개발자 도구'의 강력한 디버깅 및 분석 능력을 부여하는 도구
- 핵심 기능
  - AI가 실제 크롬 브라우저를 제어하며 실시간으로 웹페이지를 검사, 디버깅, 성능 분석 수행
  - 네트워크 요청 분석, 콘솔 로그 확인, 성능 추적 등을 통해 '눈으로 보면서' 코드를 수정하고 검증
  - 단순 코드 생성을 넘어, 생성된 코드의 실제 동작을 확인하고 최적화하는 동적인 개발 지원
- 주요 용도: 프론트엔드 코드 디버깅, 웹 성능 최적화, 복잡한 사용자 상호작용 테스트
